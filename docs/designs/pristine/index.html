About this template

Inspired by the autonomous, this template aims to be clean and simple to use (hence the "Pristine" name) and - unlike the source of its inspiration - to validate against XHTML 1.1 Strict and CSS. I have tested against Windows versions of Firefox 1.5, Internet Explorer 6, and Opera 9 - there are few very slight positioning variations but otherwise the design is still perfectly usable.

This template makes use of the excellent FamFamFam icons available at http://www.famfamfam.com/. Because Internet Explorer 6- doesn't support transparent PNG graphics, I've converted the images to JPEGs; they are still only about 600bytes each so there is no size problem.
